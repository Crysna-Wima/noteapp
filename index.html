Aplikasi ini butuh Supabase untuk menyimpan catatan. Masukkan kredensial Supabase-mu di bawah.

üìñ Cara setup Supabase (klik untuk buka)

1. Buka supabase.com → buat akun gratis → buat project baru.

2. Buka Settings → API → copy Project URL dan anon public key.

3. Buka Authentication → Providers → aktifkan Google (butuh Google OAuth Client ID dari Google Cloud Console).

4. Di Authentication → URL Configuration, tambahkan URL GitHub Pages-mu sebagai Redirect URL:
https://username.github.io/repo-name/

5. Buka SQL Editor dan jalankan SQL berikut:

-- Buat tabel notes
CREATE TABLE notes (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id) ON DELETE CASCADE,
  title TEXT DEFAULT 'Catatan Baru',
  content TEXT DEFAULT '',
  color TEXT DEFAULT 'note-yellow',
  pos_x FLOAT DEFAULT 100,
  pos_y FLOAT DEFAULT 100,
  width FLOAT DEFAULT 240,
  height FLOAT DEFAULT 180,
  z_index INT DEFAULT 1,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Aktifkan Row Level Security
ALTER TABLE notes ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Users can view own notes"
  ON notes FOR SELECT USING (auth.uid() = user_id);

CREATE POLICY "Users can create own notes"
  ON notes FOR INSERT WITH CHECK (auth.uid() = user_id);

CREATE POLICY "Users can update own notes"
  ON notes FOR UPDATE USING (auth.uid() = user_id);

CREATE POLICY "Users can delete own notes"
  ON notes FOR DELETE USING (auth.uid() = user_id);
